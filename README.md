# Kiwumil (キューミル)

Kiwumil is a TypeScript library for writing UML-style diagrams with constraint-based layout hints powered by the Kiwi solver.

For full documentation (bilingual):

- Design (English): docs/design/index.md
- 設計 (日本語): docs/design/index.ja.md
- Guidelines (English): docs/guidelines/index.md
- ガイドライン (日本語): docs/guidelines/index.ja.md

Examples: see the `examples/` directory.

Quick install:

```bash
bun install @tinsep19/kiwumil
```

Development logs: docs/devlog/ (作業ログは日本語で記録されています)
